# housespendings


- cada mes é um Folder
- cada tipo de despesa é um Label
- para cada Label tens um ficheiro dentro da Folder desse mes
- para certas Labels tens orçamemento (saude exclusive, por exemplo)
- quando consultas/ adicionas a uma Label com  orçamento:
	- avisa % do budget (" este mes gastaste x e falta y para o budget)
	- se continuares assim, no final do budget vais estar a 300% do budget
- ficheiro com macros (SALARY_JOAO, SALARY_MENA, 
	- SAVINGS_PERCENTAGE
	- LABEL_PERCENTAGE : divide-se em percentagem por label quando se quer gastar
